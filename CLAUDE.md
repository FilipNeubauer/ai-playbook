# AI Playbook — Claude Instructions

This repo is a reference library, not a runnable project.

## Structure

- `shared/` — Rules and skills that apply across all stacks
- `nestjs/` — NestJS-specific rules, project structure, and skills
- `nextjs/` — Next.js-specific rules, project structure, and skills
- `react-native/` — React Native Expo-specific rules, project structure, and skills

## Conventions

- `rules/` — directory of rule files, each covering a specific concern (e.g., `nestjs-code-rules.md`, `nestjs-folder-structure-rules.md`)
- `skills/` — Claude skills and prompt templates (e.g., `drizzle-migration.md`)
